# CS2 Tactical Shooter (Top-Down)

Un jeu de tir tactique en vue de dessus, rapide et léger, jouable directement dans le navigateur. Inspiré par l'univers de CS, avec un système d'achat, de gestion d'IA et un éditeur de cartes.

## Comment jouer ?
* **Z/Q/S/D** : Déplacements
* **Souris** : Viser et tirer
* **R** : Recharger
* **B** : Menu d'achat (dans la zone bleue)
* **1-6** : Changer d'arme

## Fonctionnalités
- IA avec machine à états (patrouille, alerte, combat);
- Moteur audio génératif;
- Système de collision fluide;
- Ultra performant;

## Installation
Pas besoin d'installation ! Ouvrez simplement `index.html` dans votre navigateur ou jouez via **GitHub Pages**.
